Frontend vs backend

Frontend: The client-side part of an application that focus on presenting stuff to the end-user

Backend: The server-side part of an application that focus on the behind-thes-scenes, logic, computations and datastorage

3 basic web technologies

  • HTML: Markup language that describes the layout and content of a static web page
  • Javascript: Client side scripting language used to add dynamic functionality to a webpage
  • CSS: Used to style a webpage
